# AQUAGEO.KZ - ReactJS pages for seminars

This is a frontend starter built with React + Vite.

## Included pages
- Home page with hero section styled to match the existing blue/lime visual identity
- Seminars catalog
- Separate seminar detail page for each course
- Seminar registration page
- Questionnaire page
- Questionnaire results page
- PDF materials viewer page
- News page with registration links

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- Data is currently mocked in `src/data/seminars.js`
- PDF preview uses `public/sample-seminar.pdf`
- You can connect forms to your backend/API later
